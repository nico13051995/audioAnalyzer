Detailed Description

Widget which displays a section of the audio waveform.

The waveform is rendered on a set of QPixmaps which form a group of tiles whose extent covers the widget. As the audio position is updated, these tiles are scrolled from left to right; when the left-most tile scrolls outside the widget, it is moved to the right end of the tile array and painted with the next section of the waveform.
